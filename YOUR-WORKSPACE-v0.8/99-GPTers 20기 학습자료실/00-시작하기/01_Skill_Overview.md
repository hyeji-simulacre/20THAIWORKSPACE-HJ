# 00. 지식 자동화 스킬 개요 (Skill Overview)

## 1. 🚀 '나만의 지식활용 AI워크스페이스'에 오신 것을 환영합니다!

우리는 **Claude Code**와 **자동화 스킬**을 활용하여 단순 반복 업무를 없애고, **진짜 중요한 일(기획, 분석, 창조)**에 집중하는 환경을 만듭니다.

### "Input → Process → Output" 자동화 공식
모든 지식 업무는 이 3단계로 이루어집니다. 우리가 배울 스킬들도 이 프레임워크 안에 있습니다.

| 단계 | 설명 | 사용할 스킬 (Skill) |
|---|---|---|
| **1. Input (수집)** | 외부 데이터를 내 컴퓨터로 가져오기 | • **유튜브 자막 추출** (영상 → 글)<br>• **PDF 리더** (잠긴 문서 → 글)<br>• **커스텀 스크래핑** (웹 → 글) |
| **2. Process (가공)** | 데이터를 정리하고 지식으로 만들기 | • **Claude Code** (AI 분석, 요약, 변환)<br>• **Python** (자동화 스크립트) |
| **3. Output (활용)** | 지식을 저장하고 공유하기 | • **Notion 연동** (지식 공유/발행)<br>• **Google Drive 연동** (자산 아카이빙) |
| **3. Output (활용)** | 지식을 저장하고 공유하기 | • **Notion 연동** (지식 공유/발행)<br>• **Google Drive 연동** (자산 아카이빙) |

---

## 2. 📂 "도구는 공장에서 일합니다" (Johnny Decimal System)

우리는 스킬(도구)을 아무 데서나 쓰지 않습니다. **정해진 폴더(공장)**에서 실행해야 지식이 쌓입니다.

| 단계 | 폴더명 | 설명 | 비유 |
|---|---|---|---|
| **수집** | **`01_inbox`** | 외부의 날것(Raw Data)을 가져오는 곳 | **하역장** (일단 쌓아두는 곳) |
| **보관** | **`50_resources`** | 가공된 지식이나 참고 자료를 모아두는 곳 | **도서관/자재창고** |
| **발송** | **`02_outbox`** | 외부(Notion, Drive)로 보낼 완성품을 두는 곳 | **출하장** (나가기 대기 중) |

> **💡 핵심 원칙**: 
> 1. **수집**한 파일은 `01_inbox`에서 `50_resources`로 이동시켜 정리합니다.
> 2. **공유**할 파일은 `02_outbox`로 복사해서 보냅니다.
> 3. **이 흐름(Flow)**이 끊기지 않는 것이 "자동화"의 핵심입니다.

## 2. 🛠️ 환경 설정 (딱 5분!)

복잡한 개발 환경 설정? NO!  
우리는 **"최소한의 설정으로 최대한의 효과"**를 냅니다.

### 1️⃣ 필수 준비물
- **VS Code**: 우리의 작업실
- **Python (3.12.4)**: 자동화 스킬을 실행하는 엔진
- **Claude Code**: 우리의 AI 파트너 (지시만 하세요)

*(이미 설치가 되어 있다면 패스!)*

### 2️⃣ 가상환경 만들기 (Virtual Environment)
내 컴퓨터의 다른 프로그램과 충돌하지 않도록, **우리만의 전용 작업 공간**을 만듭니다.

VS Code 터미널(`Ctrl + ` `)을 열고 아래 명령어를 **복사/붙여넣기** 하세요.

```powershell
# 1. 1my_1stAI_Agent 폴더로 이동 (이미 있다면 생략)
cd D:\1my_1stAI_Agent

# 2. 가상환경(.venv) 만들기
python -m venv .venv

# 3. 가상환경 켜기 (활성화)
.\.venv\Scripts\Activate.ps1
```

> **Tip**: 터미널 앞에 `(.venv)`라고 초록색 글씨가 보이면 성공입니다!

### 3️⃣ 스킬 장착하기 (설치)
우리가 사용할 자동화 도구들을 한 번에 설치합니다.

```powershell
# 이 명령어를 입력하면 필요한 모든 도구가 설치됩니다.
pip install -r .claude/skills/_education_skills/README_requirements.txt
```
*(참고: 만약 에러가 난다면, 각 스킬별 강의 자료에 있는 개별 설치 명령어를 사용하면 됩니다.)*

## 3. 💡 자동화 스킬 사용 원칙

1. **"복잡한 코드는 AI에게 맡기세요"**
   - 여러분은 코드를 짤 필요가 없습니다. *"~해줘"*라고 명령만 하세요.
2. **"레고 블록처럼 조립하세요"**
   - 수집 스킬 + 노션 업로드 스킬 = **뉴스 자동 수집 및 스크랩 시스템**
   - 유튜브 스킬 + PDF 스킬 = **멀티미디어 학습 자료 생성기**
3. **"안전제일"**
   - 내 컴퓨터의 자료는 소중합니다. 모든 스킬은 안전하게 동작하도록 설계되었습니다.
