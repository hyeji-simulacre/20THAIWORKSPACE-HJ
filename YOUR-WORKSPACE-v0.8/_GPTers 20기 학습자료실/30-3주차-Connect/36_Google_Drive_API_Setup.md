# 05. 구글 드라이브 API 설정 완벽 가이드 - "딱 한 번만 하면 됩니다!"

> **"검은 화면(터미널)보다 이 웹페이지(GCP)가 더 무섭다구요? 걱정 마세요. 그림 보고 클릭만 하면 끝납니다."**

## 1. 🎯 들어가기 힘든 첫걸음

파이썬으로 구글 드라이브를 제어하려면 **"허가증(API Key)"**이 필요합니다. 구글은 보안이 철저해서 이 허가증을 발급받는 과정이 조금 까다롭습니다.

하지만 딱 **한 번**만 해두면 됩니다. 심호흡 한번 하고 시작해볼까요? 😌

---

## 2. 🚦 5단계로 끝내는 설정 (Step-by-Step)

### Step 1: 프로젝트 만들기 🏗️
1. **[Google Cloud Console](https://console.cloud.google.com/)**에 접속하고 로그인합니다.
2. 왼쪽 위 모서리에 있는 **프로젝트 선택** (또는 `My Project` 등) 버튼을 누릅니다.
3. 팝업창 오른쪽 위의 **`새 프로젝트(New Project)`**를 클릭합니다.
4. **프로젝트 이름**: `My Drive Automation` (원하는 대로)
5. **`만들기(Create)`** 버튼 클릭! (잠시 기다리면 알림이 뜹니다. "프로젝트 선택"을 눌러주세요)

### Step 2: API 켜기 🔌
1. 왼쪽 메뉴(햄버거 버튼 ☰) > **`API 및 서비스`** > **`라이브러리(Library)`** 로 갑니다.
2. 검색창에 **`Google Drive API`** 라고 입력합니다.
3. 결과에 나온 **Google Drive API**를 클릭합니다.
4. 파란색 **`사용(Enable)`** 버튼을 클릭합니다.

### Step 3: 동의 화면 설정 (제일 중요! ⭐)
1. 왼쪽 메뉴에서 **`사용자 인증 정보(Credentials)`** > **`동의 화면 구성(Configure Consent Screen)`** 클릭.
2. **User Type**: **`외부(External)`** 선택 후 `만들기` 클릭.
3. **앱 정보 입력**:
    - **앱 이름**: `My Drive App`
    - **사용자 지원 이메일**: 본인 이메일 선택
    - (맨 아래) **개발자 연락처 정보**: 본인 이메일 입력
    - `저장 후 계속(Save and Continue)` 클릭
4. **범위(Scopes)**: 그냥 맨 아래 `저장 후 계속` 클릭.
5. **테스트 사용자 (Test Users) ⭐⭐⭐**:
    - **여기서 본인 이메일을 꼭 등록해야 합니다!**
    - `+ ADD USERS` 클릭 > **본인 구글 이메일 주소** 입력 > `추가` > `저장 후 계속`.
    - *(이걸 안 하면 "승인되지 않은 앱"이라며 로그인이 안 됩니다)*

### Step 4: 허가증(Credential) 만들기 🔑
1. 다시 왼쪽 메뉴 **`사용자 인증 정보(Credentials)`** 클릭.
2. 상단 **`+ 사용자 인증 정보 만들기`** > **`OAuth 클라이언트 ID`** 선택.
3. **애플리케이션 유형**: **`데스크톱 앱(Desktop App)`** 선택.
4. **이름**: `My Desktop Client` (아무거나)
5. **`만들기(Create)`** 클릭!

### Step 5: 다운로드 및 배치 📥
1. "OAuth 클라이언트가 생성됨" 팝업이 뜹니다.
2. **JSON 다운로드** (화살표 내려가는 아이콘 ⬇️)를 클릭합니다.
3. 다운로드된 파일(`client_secret_xxxxx.json`)의 이름을 **`credentials.json`** 으로 바꿉니다. (오타 주의! `s` 붙여야 해요)
4. 이 파일을 **우리가 작업 중인 폴더(프로젝트 최상위 폴더)**에 복사해 넣습니다.
