### 워크스페이스 다운로드

**방법 1: ZIP 다운로드 (초보자 추천)**
1. [GitHub 레포 링크](https://github.com/hyeji-simulacre/20THAIWORKSPACE-HJ) 접속
2. 녹색 `<> Code` 버튼 클릭 → `Download ZIP` 클릭 또는 Git Clone Repo
3. 압축 해제 후 원하는 위치로 이동

**방법 2: Git Clone (터미널 사용)**
```powershell
git clone https://github.com/hyeji-simulacre/20THAIWORKSPACE-HJ.git
```

### 가상환경 설정

```powershell
# 워크스페이스 폴더로 이동
cd YOUR-WORKSPACE

# 가상환경 생성
python -m venv .venv

# 활성화 (Windows)
.\.venv\Scripts\Activate.ps1

# 활성화 (Mac/Linux)
source .venv/bin/activate
```


### API 설정 필요

| 서비스 | 필요한 것 | 참고 문서 |
|--------|----------|----------|
| Google Gemini | API Key (무료) | [41_PKM_시맨틱_검색.md](40-4주차-Expand/41_PKM_시맨틱_검색.md) |

```powershell
# API 키 환경변수 설정 (Windows)
$env:GEMINI_API_KEY='your-api-key-here'

# API 키 환경변수 설정 (Mac/Linux)
export GEMINI_API_KEY='your-api-key-here'
```

### 초기 설정

```powershell
# PKM 검색 초기화
cd .claude/skills/pkm-search
python init.py    # Store 생성
python sync.py    # 파일 동기화
```
