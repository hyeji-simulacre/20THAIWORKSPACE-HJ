# 04. 노션 연동 (Notion Integration) - 옵시디언 플러그인편

> **UX적으로 좋지 않아 저는 절대로 추천하지 않습니다.**

> **이런 방법이 있다 정도로만 이해하시거나 패스하시는 것을 추천드립니다.**

> 코딩이 안 들어간다는 장점, UI로 쉽게 보여지는 장점.
> 하지만 2번이상 사용하면 오히려 공유관리가 안됩니다.

---

## 1. 🎯 개념 및 특징 (Concept)

### 🧐 파이썬 스크립트 방식과 무엇이 다른가요?
우리가 앞서 배운 파이썬 방식(`share_notion_lv1`)은 **"폴더째로 자동화"** 하는 것이 강력한 장점입니다. 하지만 설정이 조금 복잡할 수 있죠.
이번에 배우는 **플러그인 방식**은 옵시디언 내에서 **"버튼 하나로 공유"** 하는 방식입니다. 훨씬 쉽고 직관적입니다.

### 💡 이런 분들에게 추천합니다

- 🐍 파이썬 설치나 코딩이 부담스러운 분
- 📱 아이패드나 모바일에서도 노션으로 보내고 싶은 분
- ✨ 내가 원하는 글만 골라서 그때그때 보내고 싶은 분

---

## 2. 🛠️ 준비물 (Preparation)

노션의 문을 열기 위한 **열쇠(Token)**와 물건을 담을 **상자(Database)**가 필요합니다.

### 1️⃣ 노션 API 토큰 발급 (열쇠)

1. [노션 내 통합 만들기](https://www.notion.so/my-integrations) 페이지로 접속합니다.
2. **`+ 새 통합 만들기`** 를 클릭합니다.
3. 이름을 입력합니다 (예: `Obsidian to Notion`).
4. **제출**을 누르면 나오는 **`시크릿 코드`**를 복사해서 메모장에 두세요. (이게 열쇠입니다! 🔑. **중요!** 다른 사람에게 노출되어서는 안됩니다.)

### 2️⃣ 노션 데이터베이스 준비 (상자)

1. 노션에서 빈 페이지를 만들고 `/데이터베이스`를 입력해서 **[인라인]** 또는 **[전체 페이지]** 데이터베이스를 만듭니다.
2. 우측 상단 `...` 메뉴 -> `연결(Connections)` -> 방금 만든 통합(`Obsidian to Notion`)을 검색해서 추가합니다. (**중요! 권한을 주는 과정입니다**)
3. 데이터베이스의 URL을 복사합니다.
   - 예: `https://www.notion.so/myworkspace/a8aec43384f4...?v=...`
   - 여기서 `a8aec43384f4...` 부분이 **데이터베이스 ID**입니다. (`?` 앞까지)

## 3. 🔌 설치 및 설정 (Installation & Setup)

옵시디언에게 열쇠와 상자 위치를 알려줄 차례입니다.

### 1️⃣ 플러그인 설치

1. 옵시디언 **설정(Settings)** > **커뮤니티 플러그인(Community plugins)** > **사용(Turn on)**을 켭니다.
2. **탐색(Browse)** 버튼을 누르고 `notion` 을 검색합니다.
  - *업로드 플러그인: `EasyChris/obsidian-to-notion` 혹은 유사한 인기 플러그인*
  - *다운로드 플러그인: `Importer` 혹은 유사한 인기 플러그인*
3. **설치(Install)** 후 **활성화(Enable)** 합니다.

### 2️⃣ 플러그인 설정

1. 설치된 플러그인의 **설정(Options)** 메뉴로 들어갑니다.
2. 아까 복사해둔 정보를 입력합니다:
   - **Notion API Secret**: (🔑 열쇠 붙여넣기)
   - **Database ID**: (📦 상자 ID 붙여넣기)
3. 설정이 올바르다면 `Connected` 혹은 성공 메시지가 뜰 것입니다.

## 4. 🚀 사용하기 (Usage)

이제 준비는 끝났습니다. 내 노트를 노션과 공유해봅시다

1. 공유하고 싶은 옵시디언 노트를 엽니다.
2. 좌측에 있는 플러그인(`다운로드표시`, `노션표시`) 버튼을 클릭합니다.
3. 다운로드는 매번 Notion API Secret 키를 넣고 새로 페이지를 선택해서 다운로드 받아야 합니다. 🎉
4. 업로드는 노션아이콘을 클릭하면 share to (별칭) 클릭하면 업로드가 됩니다. 단 반복할 수 없습니다.! 🎉

### ❓ 혹시 안 되나요? (Troubleshooting)

가장 많이 하는 실수 2가지를 체크해보세요.

- **Check 1**: 노션 데이터베이스 우측 상단 `...` > `연결(Connections)`에 내 통합 앱이 추가되어 있나요? (이걸 안 하면 노션이 문을 안 열어줍니다!)
- **Check 2**: 토큰과 데이터베이스 ID를 복사할 때 공백(스페이스바)이 포함되지 않았나요?

## 5. ⚖️ 비교: 어떤 걸 써야 하나요?

| 특징 | 🔌 플러그인 방식 | 🐍 파이썬 스크립트 방식 |
| :--- | :--- | :--- |
| **난이도** | 쉬움 (⭐⭐) | 보통 (⭐⭐⭐) |
| **설치** | 옵시디언 내부에서 해결 | 파이썬 설치 필요 |
| **자동화** | 수동 (한 건씩 실행) | 자동 (폴더 전체 처리) |
| **유연성** | 플러그인 기능에 의존 | 코드 수정으로 무한 확장이 가능 |
| **추천** | **초보자, 개별 문서 공유** | **대량 문서 관리, 시스템 구축** |


> **Tip**: **하지만 플러그인보다 파이썬 스크립트 방식의 스킬 사용을 추천드립니다.**
